Técnico de Informática / Senac Vitória

2018 - 2020

Durante o curso Técnico de Informática no Senac Vitória, aprendi algoritmos, lógica de programação, programação web e outras habilidades fundamentais em TI. Como parte das atividades práticas, meu TCC eu desenvolvi em Java Web chamado Caderneta Virtual, um sistema online voltado para o gerenciamento de vacinas.

Destaques do projeto:

  • Sistema de segurança: Implementação de um controle de acesso robusto, redirecionando usuários sem permissões adequadas ao tentar acessar áreas administrativas.
  • Sessões HTTP: Uso de sessões para manter o estado autenticado do usuário durante a navegação, garantindo uma experiência contínua e segura.
  • Administração centralizada: O administrador podia cadastrar e gerenciar dados como vacinas, locais de vacinação e históricos de imunização.

Esse projeto, além de ser tecnicamente desafiador, me proporcionou uma base sólida em lógica de programação e desenvolvimento web, capacitando-me para resolver problemas de forma eficiente e aplicar boas práticas no desenvolvimento de sistemas.
